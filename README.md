# הטיל בדרך! (The Missile is On Its Way!)

A satirical Wolt-style "delivery tracking" dashboard for Israeli missile alerts, powered by real Pikud HaOref data.

Built with React, Tailwind CSS, Hono, and Cloudflare Workers. Costs $0 to run.
